Mission 2 : Exploitation via Interface Low-Code

Exploitation de la BDD KDou (Access QBE).

Contexte

L’objectif de cette mission est d’entamer l’exploitation de la base de données de l’entreprise KDou afin de préparer la production d’analyses et d’indicateurs fiables. Pour cela, nous allons utiliser le logiciel Low Code Access afin de réaliser différentes requêtes et commencer à structurer les informations de manière plus exploitable.
