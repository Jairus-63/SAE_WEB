Introduction au Projet

L'objectif de cette SAE était d'exploiter la base de données de l'entreprise KDou pour produire des indicateurs pertinents. Au cours de ce semestre, nous avons travaillé sur trois missions successives :

  • L'étude et l'analyse des données existantes (Mission 1).
  • L'exploitation de la base de données (Mission 2).
  • L'analyse de la nouvelle version de la base de données à l'aide du langage SQL (Mission 3).

Ce site Web sert de compte-rendu pour le module R1.02 et synthétise l'ensemble de notre travail.
